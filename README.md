## Getting Started

### Dependencies

* Node
* Android Studio
* Python 
* Expo React Native
* Django
* Postgresql

### Installing

* Frontend
```sh
cd frontend && npm install
npm run android
```

* Backend
    1. Create Python Virtual Environment
        * Windows:
        ```sh
        python -m venv env
        env/Scripts/activate.bat
        ``` 
        * Mac:
        ```sh
        python3 -m venv env
        source env/bin/activate
        ```
    2. Install Dependencies
    ```sh
    cd backend
    pip install -r requirements.txt
    ```
    3. Run Django server
    ```sh
    python manage.py runserver
    ```
